Menu creation app

This web application was created for a company called SofiMarten. They are a fast-growing company in Sofia, Bulgaria that has one takeaway food restaurant and a cake factory.

The application is about lunch menu creation. Because of their big range of lunch meals they have a menu that is changing every day. They had a willingness to share this menu with customers on their website. Basically what the application is doing is to create a table with all the selected products and then export it as a CSV file and that file can be upload on their website afterward. Other features are adding new meals into the categories, and also removing or changing prices and quantity. I also created their website and you can have a look at it on you can do on www.food.sofimarten.com but do not forget to take your google translator with you. See the project on GitHub.

Fingerprint attendance system

FingerIn SignIn and Out system is based on a web server and its working with Arduino Uno Wi-Fi to communicate with the system. The idea is to create an application that can be used by big companies to track their employees, subcontractors, or visitors. The system will automatically save time in and out and calculate working hours for each person. Also, the live statistic will be available at any time plus reports for each person will be sent by email automatically at the end of the week.

Inspired by the construction site industry I decided to build this project which will allow many companies to have a sign in and out system which is web-based and can be controlled online. I'm planning more features and if you like this project you can follow it up on GitHub.
